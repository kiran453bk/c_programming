# C Programming Repository

This repository contains a structured collection of C programs organized topic-wise and week-wise.  
It covers fundamental language constructs, control structures, modular programming, recursion, file operations, and small system-based implementations.

The repository demonstrates practical implementation of core C concepts with clean, compilable source files.
